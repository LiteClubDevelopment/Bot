# Bot

Simple Discord Bot Status Plugin

# Features:
* Customizable Status (Online/Idle/Do Not Disturb/Invisible)
* Customizable Activity Type (Playing, Competing, Streaming and More)

## Dependencies:
* Required Libraries: Spigot 1.8
